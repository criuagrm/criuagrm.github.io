# Prototipo web FCJPSRRII - UAGRM

Frontend estático multipágina para la página informativa de la Facultad de Ciencias Jurídicas, Políticas, Sociales y Relaciones Internacionales.

## Archivos

- `index.html`: portada del sitio.
- `institucional.html`: misión, visión, historia, principios, valores, decanato, vicedecanato y Consejo Facultativo.
- `carreras.html`: Derecho, Trabajo Social, Ciencia Política y Administración Pública, Relaciones Internacionales.
- `investigacion.html`: líneas, centros y unidades de investigación.
- `servicios.html`: posgrado, consultorio jurídico, biblioteca, modalidades de graduación y recursos.
- `instituto.html`: Instituto de Investigación “Dra. Mónica Von Borries”, revista Criterio Académico y galería.
- `contacto.html`: dirección, teléfonos, correo y enlaces.
- `assets/css/styles.css`: estilos visuales, responsive y paleta UAGRM.
- `assets/js/app.js`: menú móvil, navegación activa e inicialización de iconos.
- `IMG_4402.PNG`: logo de la facultad.

## Cómo editar contenido

El texto visible está directamente en cada HTML. La estructura está preparada para migrar luego a datos JSON, React o un CMS si quieres que noticias, comunicados y galerías sean administrables sin tocar código.

## Datos que faltan para la siguiente version

- Logo oficial o sello que quieras usar.
- Fotografías reales de la facultad y de cada carrera.
- Autoridades completas, oficinas, teléfonos, correos y horarios.
- Enlaces oficiales y redes sociales.
- Noticias, eventos, comunicados, resoluciones, convenios y documentos.
- Mallas curriculares, perfiles profesionales y requisitos por carrera.
